## 📌 Project Description
This is a web-based system developed to manage products and services for Ian Industries and Repairs.  
The system allows users to:
- Log in securely  
- View available products  
- Access services offered  
- Contact the business for more information and services offered 

---

## ⚙️ Technologies Used
- PHP  
- MySQL  
- HTML  
- CSS  
- JavaScript  
- XAMPP  

---

## 🚀 How to Run the Project

1. Install XAMPP  
2. Start **Apache** and **MySQL**  
3. Copy the project folder to:
   D:\xampp\htdocs\
4. Open browser and go to:
   http://localhost/wasswa_ian

---

## 🗄️ Database Import Instructions

1. Open phpMyAdmin  
2. Create a new database (e.g. `Wasswa_ian`)  
3. Click **Import**  
4. Select the `.sql` file from the project  
5. Click **Go**
